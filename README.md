# Installation

- `git clone git@gitlab.com:PeterDKC/cat-friends.git`
- `cd cat-friends`
- `composer install`
- Copy `.env.example` to `.env`, set any necessary details such as your database user and password.
- `php artisan key:generate`
- `php artisan cats:fresh`
